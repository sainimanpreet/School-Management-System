# 🎓 School Management System

A full-stack School Management System built using Spring Boot and React.js. This application helps manage students, teachers, courses, and administrative operations through a modern web interface.

## 🚀 Features

- 👨‍🎓 Student Management
  - Add, update, delete, and view student records

- 👩‍🏫 Teacher Management
  - Manage teacher information and profiles

- 📚 Course Management
  - Create and manage courses

- 🔍 Search & Filter
  - Quickly find students, teachers, or courses

- 📊 Dashboard
  - Centralized view of school data

- 🔐 REST API Integration
  - Spring Boot backend connected with React frontend

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/sainimanpreet/School-Management-System.git
cd School-Management-System
```

### Backend Setup

```bash
mvn spring-boot:run
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
